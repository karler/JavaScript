Ejercicio 05

Solución del ejercicio 1

Crea una aplicación que reciba un producto y devuelva el porcentaje de descuento aplicado según las siguientes reglas:

  • Si el producto es "camisa", el descuento es del 10%.
  • Si el producto es "pantalon", el descuento es del 15%.
  • Si el producto es "vestido", el descuento es del 20%.
  • Si el producto es "zapatos", el descuento es del 25%.
  • Si el producto no se encuentra en la lista, no se aplica ningún descuento (0%).
