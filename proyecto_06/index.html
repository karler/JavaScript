Ejercicio 06

Solución del Ejercicio 2

Escribe una aplicación que reciba un mes y devuelva la estación del año correspondiente. Utiliza la estructura switch para resolver este ejercicio.

Ejemplo:

  • Si el mes es "enero", "febrero" o "diciembre", la función debe devolver "Invierno".
  • Si el mes es "marzo", "abril" o "mayo", la función debe devolver "Primavera".
  • Si el mes es "junio", "julio" o "agosto", la función debe devolver "Verano".
  • Si el mes es "septiembre", "octubre" o "noviembre", la función debe devolver "Otoño".
  • Si el mes no se encuentra en la lista, la función debe devolver "Mes inválido".
