Solución del Ejercicio 8

Acceso al sistema:

Pide al usuario su nombre de usuario y contraseña. Define dos variables con las credenciales correctas: usuarioCorrecto y contraseñaCorrecta. Si el usuario ingresa un nombre de usuario diferente al usuarioCorrecto O una contraseña diferente a la contraseñaCorrecta, mostrar "Acceso denegado". Si ambas coinciden, mostrar "Acceso concedido".
